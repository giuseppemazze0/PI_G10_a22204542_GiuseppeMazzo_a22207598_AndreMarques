# Estrutura do projeto

README.md
- fiheiro com a tabela que descreve o projeto e os alunos

\src
- pasta com o código do projeto

\model 
- pasta com ficheiro model.md de outros ficheiros relacionados/revelantes.

\docs
- pasta com a documentação HTML gerada a partir do Javadoc

\logs
- logs de prompts usadas durante as aulas
- prompts que tenham resultados interessantes/surpreendentes (pela positiva ou pela negativa)
- cada log deve ser colocado num ficheiro .txt diferente
